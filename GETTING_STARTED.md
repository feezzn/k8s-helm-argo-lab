# Getting Started

Bem-vindo ao k8s-helm-argo-lab! 

Este laboratório foi estruturado pra você aprender de forma prática e progressiva.

## 🚀 Primeiros passos

1. **Leia** [README.md](../README.md) pra entender a estrutura
2. **Siga** o setup em [01-setup/README.md](../01-setup/README.md)

- **01-setup/** - Como instalar e configurar tudo

- **04-terraform/** - Infrastructure as Code

- **06-projects/** - Projetos práticos completos

## 🎯 Objetivo

Ao final deste laboratório você saberá:
- ✅ Deploy Kubernetes local com Minikube
- ✅ Usar Helm pra packagear aplicações

- ✅ Infraestrutura como código com Terraform
- ✅ Automação com Ansible
